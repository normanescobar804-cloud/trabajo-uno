# El Close de Jiet

Sitio web estático para la tienda "El Close de Jiet".

Contenido:
- `index.html` — página principal
- `style.css` — estilos
- `script.js` — lógicas de carrito y búsqueda
- `product.html`, `product.js` — páginas individuales de producto

Instrucciones rápidas:
1. Abrir `index.html` en un navegador.
2. Para subir a GitHub: inicializar un repositorio, agregar remote y hacer push.
